# Animal Re-Identification using Deep Metric Learning

##  Overview
This project focuses on identifying and clustering individual animals from images using deep metric learning in a semi-supervised open-set setting.

##  Key Features
- Transformer-based feature extraction (MegaDescriptor-L)
- Metric learning with cosine similarity
- Unknown identity discovery using DBSCAN
- Threshold optimization using ARI

##  Results
- **ARI Score:** 0.6

##  Model Pipeline
Input Image → Feature Extraction → Embedding → Similarity Matching / Clustering
